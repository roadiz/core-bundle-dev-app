# Optional Roadiz Bundles

Roadiz provides several optional bundles that extend the core functionality with specialized features. These bundles are separate packages that you can install and configure based on your project requirements.

## Available Optional Bundles

### [Solr Bundle](./solr-bundle.md)
Full-text search engine integration using Apache Solr for indexing and searching NodesSources and Documents.

### [Two-Factor Authentication Bundle](./two-factor-bundle.md)
Two-factor authentication system for enhanced security, supporting OTP authentication with Google Authenticator, backup codes, and trusted devices.

### [User Bundle](./user-bundle.md)
Public user management system for handling user registration, authentication, password reset, and account management for front-end users.

## Installation

Each optional bundle can be installed independently using Composer. Refer to the individual bundle documentation pages for detailed installation and configuration instructions.

::: tip
These bundles are completely optional and can be installed only if your project requires their specific functionality.
:::
