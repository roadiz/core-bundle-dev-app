## Contribute

To enhance Rozier backend theme you must install Grunt and Bower:

```shell
cd src
yarn install
# Launch Grunt to generate prod files
yarn build
# Or… launch watch grunt when you’re
# working on LESS and JS files.
yarn dev
```
